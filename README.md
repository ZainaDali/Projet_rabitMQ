# 🐰 Projet RabbitMQ – Calculs distribués avec interface Tkinter

## 🎯 Objectif

Ce projet implémente un système de calcul distribué avec **RabbitMQ** et **Python**. Il comprend :
- Un **client** qui envoie des requêtes de calcul (addition, soustraction, multiplication, division).
- Plusieurs **workers**, chacun spécialisé dans un type d’opération.
- Un **récepteur de résultats**.
- Une **interface graphique Tkinter** pour tout contrôler et visualiser.
